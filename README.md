# Vora Website

🔗 Live: https://vorabusinessplc.vercel.app/

---

## Overview

This is a responsive business website built to showcase services and collect customer inquiries through a contact form.

The project focuses on simplicity, clean UI, and functional form handling without a backend.

---

## Features

* Responsive design (mobile and desktop)
* Contact form with Formspree integration
* JavaScript-based form submission (async)
* Success and error feedback
* URL parameter message autofill
* Mobile navigation (hamburger menu)

---

## Tech Stack

* HTML5
* Tailwind CSS
* JavaScript (Vanilla)
* Formspree
* Vercel (deployment)

---

## How It Works

1. User fills out the contact form
2. Form data is sent to Formspree
3. A success or error message is displayed
4. The form resets after submission
